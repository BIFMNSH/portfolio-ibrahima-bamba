# Impeccable — Portfolio

Ce projet utilise Impeccable comme cadre d'audit design pour éviter les interfaces génériques et améliorer la qualité visuelle du portfolio.

## Commandes utiles

Dans le terminal du projet :

```bash
npx impeccable detect .
```

Dans un agent IA compatible, par exemple Codex CLI :

```bash
codex
```

Puis dans Codex :

```text
/impeccable live
```

## Objectif design

- Rendu premium sobre.
- Hiérarchie typographique claire.
- Animations discrètes.
- Positionnement recruteur immédiatement lisible.
- Mise en avant du cas copropriété.

## À vérifier avant chaque push

```bash
npx impeccable detect .
git status
```
